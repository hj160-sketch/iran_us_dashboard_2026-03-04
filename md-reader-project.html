MD Reader

A reading and note-taking tool for Markdown-heavy workflows, designed to make long-form material easier to review, extract, and turn into structured notes.

Input

Markdown, TXT, PDF, EPUB

The reader accepts multiple file types and can use PDF OCR when needed, which makes it usable for both clean text and scanned documents.

Workflow

Sentence to Notes

The core interaction is simple: pick a sentence on the left, move it into notes on the right, and keep building a clean reading trail.

Accessibility

ADHD-Friendly Mode

The app includes a focused reading mode with larger type and looser line spacing for lower-friction review and annotation.
