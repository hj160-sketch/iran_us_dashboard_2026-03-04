# US-Iran Interactive Dashboard

An interactive market-and-geopolitics dashboard focused on the U.S.-Iran situation, with multi-asset charting, event-driven news linkage, scenario lines, and Dalio-style historical comparisons.

## Live Links

- Project page: <https://hj160-sketch.github.io/iran_us_dashboard_2026-03-04/>
- Personal homepage: <https://hj160-sketch.github.io/>
- Repository: <https://github.com/hj160-sketch/iran_us_dashboard_2026-03-04>

## Main Features

- Multi-asset market board:
  - U.S. broad market proxies (`SPY`, `QQQ`, `DIA`)
  - Oil proxy (`USO`)
  - Precious/industrial metals (`GLD`, `SLV`, `CPER`)
  - Defense sector and key names (`ITA`, `RTX`)
  - Korea market proxy and Samsung (`EWY`, `005930.KS`)
- Interactive K-line (candlestick) panel:
  - Time ranges: `5m`, `3H`, `1D`, `1W`, `1M`, `3M`, `6M`, `1Y`, `ALL`
  - Multi-asset overlay lines
  - Three scenario lines (base/escalation/de-escalation)
- Event timeline and news details:
  - Major events, policy moves, sanctions, military updates, and social-post signals
  - Click-driven linkage between chart points and event/news cards
- News heat and forecast block:
  - News heat dots with click-to-sync event details
  - Forecast cards (`7D` / `30D`) with UP probability and T+1/T+5 summary
- Dalio framework panel:
  - Historical analog cases
  - Linked case chart, references, and explanatory notes

## Local Usage

This project is a static HTML dashboard.

1. Open `index.html` directly in a browser, or
2. Open `iran_us_interactive_dashboard.html` directly in a browser.

No build step is required.

## Notes

- Data in this version is dashboard-oriented and partially model/simulation-based for interactive analysis workflows.
- It is designed for research and scenario discussion, not investment advice.

# 美国-伊朗交互看板

这是一个围绕美国-伊朗局势的交互式市场与地缘事件看板，支持多资产图表、事件联动新闻、情景线推演，以及达里奥框架下的历史对标。

## 在线地址

- 项目页面：<https://hj160-sketch.github.io/iran_us_dashboard_2026-03-04/>
- 个人主页：<https://hj160-sketch.github.io/>
- 仓库地址：<https://github.com/hj160-sketch/iran_us_dashboard_2026-03-04>

## 核心功能

- 多资产市场看板：
  - 美股大盘代理（`SPY`、`QQQ`、`DIA`）
  - 油价代理（`USO`）
  - 贵金属/工业金属（`GLD`、`SLV`、`CPER`）
  - 军工板块与重点个股（`ITA`、`RTX`）
  - 韩国市场代理与三星（`EWY`、`005930.KS`）
- 交互 K 线主图：
  - 时间区间：`5m`、`3H`、`1D`、`1W`、`1M`、`3M`、`6M`、`1Y`、`ALL`
  - 多资产叠加对比
  - 三种未来情景线（基准/升级/缓和）
- 重大事件与新闻联动：
  - 重大事件、政策、制裁、军事、社媒信号
  - 点击图表与时间线可联动到事件详情与新闻
- 新闻热度与预测模块：
  - 热度点阵可点击联动
  - 预测卡支持 `7D` / `30D`、UP 概率、T+1/T+5 概览
- 达里奥框架区域：
  - 历史案例对标
  - 案例K线、章节页码与说明文字联动

## 本地使用

这是静态 HTML 项目。

1. 直接用浏览器打开 `index.html`，或
2. 直接用浏览器打开 `iran_us_interactive_dashboard.html`。

无需构建步骤。

## 说明

- 当前版本数据用于交互分析展示，部分为模型/模拟口径以支持场景推演。
- 内容用于研究讨论，不构成投资建议。
